This directory contains fully implemented and closed planning documents for the AI Orchestrator project. Each file represents a feature or refactor that has been designed, implemented, verified, and committed: `bigchange_rtk_integration_Completed.md` covers the RTK (Remote Token Key) integration overhaul; `claude_completed.md`, `claude2_completed.md`, and `claude3_completed.md` are successive improvement rounds for the Claude CLI adapter and orchestration pipeline; `copilot_completed.md` and `copilot-t3code_completed.md` document Copilot CLI adapter work and the T3-code variant integration; `cursor_completed.md` covers Cursor CLI adapter integration; `gemini_completed.md` covers the Gemini CLI adapter; `memory-research_completed.md` documents the memory subsystem research and implementation; `plan_loop_mode_Completed.md` describes the loop-mode coordination feature; and `unified_plan_completed.md` is the original unified architecture consolidation plan. No file in this directory should be modified — if follow-up work is needed, create a new dated plan in the parent `docs/plans/` directory.
